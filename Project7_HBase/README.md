#### 3.Mysql迁移至HBase
	将Mysql某一数据库中的所有表及视图迁移至HBase
从MySQL导入到HBase中，可以实现一对一，也可以实现多对一
   问：什么场景下使用多对一？
   答：关联查询非常多的情况下
   问：如何实现多对一？
   答：在HBase中，(1)一个Mysql表对应一个列族;(2)所有表的字段都作为列族的列存在
     在MySQL中，在MySQL中，建立一个视图，转化为一对一的实现方式。
